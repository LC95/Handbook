- Collections 集合
- Configuration 配置项

- Json Json处理
- Linq Linq处理
- Localization 本地化
- Logging 日志
- Modules 模块

- ObjectMapping 物体映射
- Orm 对象关系映射

- Realtime 实时 
- Reflection 反射

- RandomHelper 随机数帮助类

- UserIdentifierExtensions 用户认证拓展
